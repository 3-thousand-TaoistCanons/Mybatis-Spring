#SSM集成的基础项目，项目使用Maven管理

项目使用Spring3.2.12+SpringMVC3.2.12+Mybatis3.2.8

项目集成了Mybatis分页插件和通用Mapper插件

项目使用的hsqldb内存数据库，根据需要可以切换为其他支持的数据库

##推荐使用Mybatis通用Mapper

项目地址:https://github.com/abel533/Mapper
